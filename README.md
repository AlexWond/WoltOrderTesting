# WoltOrderTesting

## Link to website [wolt.com](https://wolt.com/)

### Technologies:

* Node.js
* Cypress

### Setup instructions:
#### 1. Navigate to [Node.js](https://nodejs.org/en) and follow instructions to download and install *Node.js* and *npm*
#### 2. Navigate to [GitHub repository](https://github.com/AlexWond/WoltOrderTesting)
#### 3. Click `Code`
#### 4. Choose `HTTPS`
#### 5. Click `Copy URL to clipboard`
#### 6. Open `Terminal` application in your operating system
#### 7. Navigate to the chosen destination folder
#### 8. Use command `git clone ` and paste HTTPS from clipboard
#### 9. Navigate to *WoltOrderTesting* folder using `cd .\WoltOrderTesting\` command
#### 10. Use `npm install` command to install packages
#### 11. Use `cypress run --headed --spec .\cypress\e2e\wolt.spec.cy.js` to run the test (with 'displayed browser' option)
